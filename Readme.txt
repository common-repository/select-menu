== Description ==

This plugin is free. And you can use this plugin to add/ create select menu from custom menu or your primary menu.
You can add the select menu from widget. This plugin especially use for responsive template that need select menu for mobile template support.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your select menu from widget
